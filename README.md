# Build A Brand

A high-conversion landing page for Build A Brand - a digital marketing agency that helps businesses turn website traffic into clients through structured digital systems.

## Overview

Build A Brand is a subsidiary of DEVYRA GROUP, specializing in:
- Conversion-focused website development
- Lead generation systems
- Marketing automation
- Digital strategy consulting

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Dark theme with gold accents for a premium look
- **SVG Graphics**: Custom SVG illustrations for hero and conversion sections
- **Smooth Animations**: CSS transitions and smooth scrolling
- **Contact Form**: Lead capture form for strategy calls

## Technologies Used

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript
- Google Fonts (Poppins)
- Font Awesome Icons

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/devyra-group-admin/buildabrand.git
   ```

2. Open `index.html` in your browser or serve with a local server:
   ```bash
   # Using Python
   python -m http.server 8080
   
   # Using Node.js
   npx serve
   ```

3. Visit `http://localhost:8080` in your browser

## Development

### CSS
The stylesheet uses CSS custom properties for easy theming. Main color variables:
- `--color-gold`: Primary brand color (#d4a532)
- `--color-bg-primary`: Main background (#0a0a0a)
- `--color-text-primary`: Primary text color (#fff)

### JavaScript
The main.js file handles:
- Smooth scrolling navigation
- Navbar background on scroll
- Form submission handling
